# Projects & Work

## Personal Website with AI Chatbot

**Current Project** - A modern portfolio website featuring an intelligent AI chatbot

### Key Features
- **RAG-powered Chatbot**: Uses ChromaDB and OpenAI to answer questions about my background
- **Modern Tech Stack**: React, Vite, Tailwind CSS, Framer Motion
- **Responsive Design**: Works seamlessly across all devices
- **Content Management**: Markdown-based writing system with automated generation
- **Interactive Elements**: Smooth animations and engaging user experience

### Technical Implementation
- **Frontend**: React with TypeScript and modern hooks
- **Styling**: Tailwind CSS with custom gradient borders and animations
- **Backend**: Vercel serverless functions for the chatbot API
- **Database**: ChromaDB for vector storage and semantic search
- **AI Integration**: OpenAI GPT-3.5-turbo with RAG (Retrieval Augmented Generation)
- **Deployment**: Vercel for hosting and automatic deployments

### Learning Outcomes
- **Full-stack Development**: End-to-end application development
- **AI Integration**: Implementing RAG systems with vector databases
- **Modern Web Technologies**: Latest React patterns and best practices
- **DevOps**: CI/CD, environment management, and deployment strategies
